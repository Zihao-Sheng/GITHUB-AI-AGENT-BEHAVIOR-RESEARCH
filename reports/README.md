# Reports

This directory contains human-readable project outputs.

| Path | Description |
| --- | --- |
| `final_report.pdf` | Final written research report. |
| `html/` | Rendered HTML notebook outputs and interactive network visualizations. |
| `figures/` | Static README-ready figures generated from processed project data. |
| `archive/` | Earlier milestone reports retained for project-history traceability. |

For a quick review, start with `final_report.pdf`, then open the rendered HTML files in `html/`.

Key static figures include `figures/repo_community_detection.png`, `figures/readme_overview.png`, and `figures/agent_language_network.png`.
